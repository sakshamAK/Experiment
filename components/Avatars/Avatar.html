Avatars
Avatar can be used to show user's profile picture on profile information page,on navigation bar, in blogs grid items.

Standard Avatar

The Avatars components has four sizes.You need to include class according to size the size of avatar you want. avatar-xl,avatar-lg,avatar-std,avatar-sm.

Avatars with Status

Avatars support presence indicators.There are three indicators online,offline and dnd avatar-xl,avatar-lg,avatar-std,avatar-sm.
